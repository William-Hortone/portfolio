HTML5

La base de tout projet web. Je comprends son fonctionnement et je l’utilise afin de créer et de représenter le contenu d’une page web et sa structure.

CSS3

J’ai des bases solides en css, je peux facilement styliser un document HTML. Par ailleurs, je connais plusieurs concepts tels que : flex box, css grid, les animations, le responsive disgn ...

JAVASCRIPT

Au cœur des langages utilisés par les développeurs web afin d’avoir des sites interactifs. J’ai su comprendre son fonctionnement et masteriser plusieurs notions.

BOOTSTRAP

Après avoir acquis les bases solides en css, j’ai appris Bootstrap qui est par ailleurs un framework css qui fait gagner du temps en évitant de devoir gérer les modeles de façon répétitive.

REACT JS

Pour enrichir mes connaissances en développement front-end, j'ai appris React js. Qui me permet de créer un code simple et intuitif, d'accélérer mon processus de développement mais aussi la rapidité des mes applications web.

GITHUB

Enfin stocker et gérer mon code, mais également dans le but de partager mon travail en le rendant accessible à tous et de travailler en équipe, j’ai appris à me servir de git et github.

C++

Etant l'un des langages de programmation les plus utilisés par les développeurs, notamment en matière d'applications. C'est l'un des langages avec lequel j'ai débuté dans l'univers du code. Cela m'a permis d'assimiler différents concepts et m'a facilité l'apprentissage de Javascript.
